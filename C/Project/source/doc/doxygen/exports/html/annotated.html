Dijkstra's algorithm project  3.0
Dijkstra's algorithm implementation to find shortest path between source and destination nodes in graph (graphics with gnuplot).
Data Structures
Here are the data structures with brief descriptions:
 CarchStruct-typedef description: Arch struct typedef to store arch name, arch cost and connection nodes associated to that arch (Node1 and Node2)
 CconnectionStruct-typedef description: Node connection, composed by connection arch and node on the other edge of the arch
 Cdate_timeStruct-typedef description: Date/time data-structure (yyyy/mn/dd, hh:mm:ss), defined using get_date_time() or assign_date_time() functions.
Date/time exact format: (yyyy) + DATE_DELIM_CHR + (mn) + DATE_DELIM_CHR + (dd) + DATE_TIME_DELIM_CHR + (hh+1) + TIME_DELIM_CHR + (mm+1) + TIME_DELIM_CHR + (ss+1)
 Cdijkstra_datasetStruct-typedef description: Dijkstra-dataset struct typedef to store informations about a specific node, in order to be able to find min-cost path with Dijkstra's algorithm and reconstruct shortest path
 Clist_elemStruct-typedef description: List element struct typedef to store connection archs associated to a specific node
 CnodeStruct-typedef description: Node struct typedef to store node name, a list of connection archs associated to that node and a Dijkstra-dataset which contains the informations about that specific node to find min-cost path with Dijkstra's algorithm and reconstruct shortest path
 CstreetStruct-typedef description: Street struct typedef for graph-library test (use crosses and streets as example of archs and nodes)
