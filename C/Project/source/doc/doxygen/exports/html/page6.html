User-interface-library info

Library description:

This library has the main purpose of managing user-interface terminal inputs/outputs (files lib_ui.h and lib_ui.c).

Library details:

Library to securely manage terminal I/O operations, define terminal colors, define aliases for standard data-types and include most used libraries.

  • Main library data-types:
    • Byte  --> unsigned char alias (Byte).
    • Str  --> char* alias (Str).
    • Cstr  --> const char* alias (Cstr).
    • C_str  --> const char* const alias (C_str).
    • Ptr  --> void* alias (Ptr).
    • C_ptr  --> const void* const alias (C_ptr).
  • Library public variables:
    • term_in_buff  --> Terminal input buffer char array for scanf function, to read inputs from user (size defined using IN_BUFF_SIZE marco).
    • unused  --> Unused variable to catch unused returns from functions.
  • Main library public functions:
    • logo()  --> Function to print responsive-logo on terminal.
    • fbk_separator()  --> Function to print responsive terminal-outputs separator.
    • press_enter()  --> Function to ask enter key in order to continue (on terminal).
    • fbk_nl()  --> Function to print new line feedback on terminal.
    • fbk_gn_pu()  --> Function to print green/purple feedback on terminal (generally used for request-feedbacks).
    • fbk_gn_cy()  --> Function to print green/cyan feedback on terminal (generally used for operation-completed feedbacks).
    • fbk_gn_lbu_ye_real()  --> Function to print green/light-blue/yellow Real feedback on terminal.
    • fbk_err()  --> Function to print red error feedback on terminal.
    • read_term_in_min_chrs()  --> Function to safely read user terminal-input.
    • read_term_in_int_inrange()  --> Function to safely read user terminal-input, making sure input value is in defined range.
    • read_term_in_confirm()  --> Function to ask confirmation in order to continue (on terminal).
    • dbg_str()  --> Function to print debug string feedback on terminal (light-green/yellow/orange colored debug feedback).
    • close_err()  --> Function to close software with error feedback printing on terminal.
    • close_fbk()  --> Function to close software with goodbye feedback printing on terminal.
    • close_keyboard_interrupt()  --> Keyboard interrupt (ctrl+c) event-handler function: close software with specific feedback.
  • Library-defined terminal colors:
    • RD  --> Red color.
    • BU  --> Blue color.
    • LBU  --> Light-blue color.
    • PU  --> Purple color.
    • CY  --> Cyan color.
    • YE  --> Yellow color.
    • OG  --> Orange color.
    • GN  --> Green color.
    • LGN  --> Light-green color.
    • LGY  --> Light-gray color.
    • ER  --> End color.
