# Project about graphs and Dijkstra's algorithm:

## <strong>Content:</strong>

___Libraries___:

  • Graph: library for graphs management and minimum cost path with Dijkstra's algorithm.
  • UI: GUI (terminal I/O) library to print logo and coloured terminal feedbacks.

___Main___

  • Graph_test: main test software to create a small graph and test Dijkstra's algorithm to find minimum cost path.

___Makefile___

  • ___Compile commands:___
    • make: command to compile and link the whole project to executable (shortcut command).
    • make all: same command as above, extended command.
    • make run: command to launch

* <strong>make all_run</strong>: command to compile/link the whole project to executable and run it.
