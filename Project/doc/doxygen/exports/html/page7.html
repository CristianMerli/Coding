7 - Files library

Files-library info

Library description:

This library has the main purpose of managing files open/close and read/write operations (files lib_files.h and lib_files.c).

Library details:

Library to manage files using <stdio.h> system-library, giving the possibility to write strings/chars, searching for matches and to read/extract data.

  • Library data-types:
    • Fl  --> File pointer data-type.
  • Library public variables:
    • file_in_buff  --> File input buffer char array for fgets function, to read each line of the file (size defined using FILE_IN_BUFF_SIZE marco).
  • Library public functions:
    • open_file()  --> Function to open a file.
    • write_int_on_file()  --> Function to write an integer on file.
    • write_str_on_file()  --> Function to write a string on file.
    • write_sep_on_file()  --> Function to write a separator char on file.
    • write_nl_on_file()  --> Function to write a new-line on file.
    • write_tab_on_file()  --> Function to write a tab char on file.
    • read_from_file()  --> Function to read from file, printing feedback on terminal.
    • get_substr_line_from_file()  --> Function to get string (or substring) line number inside file.
    • get_line_str_from_file()  --> Function to get line string content from file.
    • close_file()  --> Function to close a file.
