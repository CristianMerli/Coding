Timer-library info

Library description:

This library has the main purpose of managing time's concerning operations (files lib_timer.h and lib_timer.c).

Library details:

Library to manage date/time data structures from <time.h> system-library and to give the possibility to wait a delay time during runtime.

  • Library data-types:
    • date_time  --> Date/time personalized data-structure.
  • Library public functions:
    • delay()  --> Function to wait specified delay time in milliseconds.
    • get_date_time()  --> Function to get current date/time.
    • assign_date_time()  --> Function to manually assign date/time values.
    • cmp_date_time()  --> Function to compare two dates/times.
    • fbk_gn_lbu_ye_date_time()  --> Function to print date/time feedback.
